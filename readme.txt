Put an xml file in a directory with executable.
Программа будет подхватывать XML из папки с бинарником.
---

Структура:

Класс Link - абстрактный класс, реализует паттерн проектирования Стратегия. 
	     Его потомки предоставляют реализацию, в то время как от Link - интерфейс.
	     обертка для указателя на абстрактный объект.
	     
	     
Класс AbstractObject - класс, который можно скомпозировать данными, что бы он был похож на "настоящий" объект из реального мира.
                       В себе содержит список свойств ( Class Property - обертка над QString) и список Связей ( Link* ).
                       Умеет вызывать интерфейс всего, до чего дотягивается.
                       
LinkFactory - Класс , реализующий паттерн проектирования Фабричный метод.
              Хранит в себе хэш-таблицу , где ключь - имя типа связи. Значение - Объекты std::function, которые инкапсулируют в себе функции, которые ответственны за создание определенных типов.
              
              
ObjectPool - хранит в себе все созданные абстрактные объекты. Является синглтоном.
